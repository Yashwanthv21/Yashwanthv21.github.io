Regression

Use the Training Data to train a model using the Features and Predict the TARGET variable.

Goal

It is your job to predict the TARGET for each row. For each Id in the test set, you must predict the TARGET value. 

Metric

Submissions are evaluated on Root-Mean-Squared-Error (RMSE) between the logarithm of the predicted value and the logarithm of the actual TARGET value.

Submission File Format

The file should contain a header and have the following format:

ID,TARGET
1,169000.1
2,187724.1233
3,175221
etc.

You can download an example submission file (sample_submission.csv)  Here.

Note: The submission file should be named as submission.csv and must be Zipped before submitting. Your submission might not be scored if it is not correctly named.
